# AI do wykrywania skoliozy ze zdjęć RTG / Scoliosis Detection AI

## Polski

### O projekcie

Stworzyłem aplikację wykorzystującą **sztuczną inteligencję** do analizy zdjęć **RTG kręgosłupa**, automatycznego wykrywania skoliozy oraz obliczania **kąta Cobba**. Użytkownik może także **ręcznie zaznaczyć linie**, aby zweryfikować lub poprawić wynik.  
Projekt powstał jako część mojej **pracy inżynierskiej**.

### Funkcjonalności

- Wgrywanie zdjęć **RTG**
- Automatyczne wykrywanie kręgosłupa (AI - **Ultralytics YOLO**)
- Obliczanie **kąta Cobba**
- Ręczne zaznaczanie linii pomiarowych
- Wizualizacja przetworzonych obrazów
- Prosta aplikacja okienkowa do obsługi

### Technologie

- **Python 3**
- **Ultralytics YOLO**
- **OpenCV**
- **NumPy**
- **Tkinter / PyQt**

## English

### About the Project

I created an application that uses **artificial intelligence** to analyze **spine X-ray images**, automatically detect scoliosis, and calculate the **Cobb angle**. The user can also **manually draw lines** to verify or correct the measurement.  
The project was developed as part of my **engineering thesis**.

### Features

- Uploading **X-ray images**
- Automatic spine detection (AI – **Ultralytics YOLO**)
- **Cobb angle** calculation
- Manual line marking
- Processed image visualization
- Simple desktop application for interaction

### Technologies

- **Python 3**
- **Ultralytics YOLO**
- **OpenCV**
- **NumPy**
- **Tkinter / PyQt**
